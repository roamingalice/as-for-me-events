ASify 年中一番賞

這是一個門市限定活動。客人到門市購買一番賞籤，每籤 NT$1,500，每籤必中，最低獎也會帶走高於抽籤金額的 ASify 商品組。活動目標是製造門市話題、協助指定品項出清，並讓客人用有趣的方式認識更多 ASify 商品。

活動地點 ASify 門市限定
購買方式 NT$1,500 / 籤，每籤必中
獎池規模 25 籤一套，售完為止

活動一句話

門市同事可以這樣跟客人介紹。

「年中一番賞門市限定，每抽 NT$1,500、每抽必中，最低也會帶走價值 NT$1,800 的 ASify 商品組；點到最後一格時，會得到原獎項並加贈 Last One 風衣任選。」

門市流程

活動執行照這五步走。先付款，再抽籤，再兌獎；每次兌換完成後，Pad 要立即更新剩餘獎項。

1

客人購買

POS 收取「年中一番賞」NT$1,500 / 籤。此商品不使用購物金、不併用折扣、不累計滿額贈。

2

開啟 Pad

店員確認付款完成後，開啟 Pad 抽籤頁，讓客人選擇籤格。

3

客人點籤

客人點選一格，畫面翻出獎項。抽出後不可重抽或更換。

4

現場兌獎

店員依畫面獎項交付商品；服飾獎項依現場尺寸、顏色與庫存兌換。

5

完成紀錄

店員按「已兌換」，Pad 更新已抽籤數、剩餘獎項與 Last One 狀態。

Last One 說明

Last One 是尾盤獎勵，規則要讓同事與客人都能一眼理解。

怎麼拿到

最後一格被點下時，該客人會得到該籤原本抽中的獎項，並加贈 Last One 風衣任選。

一次買多張怎麼算

如果客人一次購買多張，且其中包含該套最後一格，Last One 由該客人獲得。

現場提醒

Pad 不需要把 Last One 做成可點的格子。剩最後一格時，店員要提醒客人「這一格會是原獎項 + Last One」。

門市 SOP

執行時只要照順序確認：付款、抽籤、兌獎、更新狀態。

店員操作

  • 確認客人購買籤數，POS 收款。
  • 開啟 Pad 抽籤頁，讓客人點選籤格。
  • 依畫面獎項交付商品。
  • 服飾獎項只依現場尺寸、顏色與庫存兌換。
  • 按「已兌換」，更新剩餘獎項。

不可做

  • 一番賞不可使用購物金。
  • 一番賞不併用折扣。
  • 一番賞不可累計滿額贈。
  • 一番賞不可換現金。
  • 抽出後不可重抽。
  • 獎品不可折現或換成其他非對應獎項。
  • 服飾獎項不事後寄送、不調貨、不提供預留。

現場話術

  • 每籤 NT$1,500，每籤必中。
  • 最低獎也是價值 NT$1,800 的商品組。
  • 獎項以現場尺寸顏色為準。
  • 最後一格會拿到原獎項，並加贈 Last One 風衣。
  • Pad 會即時顯示剩餘獎項。

現場公告規則

這些內容建議放在 Pad 頁面與門市告示，避免現場認知落差。

  • 本活動為 ASify 門市限定，每籤 NT$1,500，每籤必中。
  • 每套共 25 籤，獎項數量與剩餘狀態依現場 Pad 顯示為準。
  • 一番賞不可累計滿額贈、不併用折扣、不使用購物金。
  • 服飾類獎項皆以門市現場可選尺寸、顏色與庫存為準，不提供寄送、調貨或預留。
  • 獎品不得兌換現金、購物金或其他非對應獎項。
  • 抽籤完成後不得重抽、退換或作廢。
  • 最後一格被點下時，該客人可獲得原本抽中的獎項，並加贈 Last One 風衣任選。

待確認

以下項目確認後，就可以進入 Pad 抽籤頁與門市物料製作。

活動期間

確認開始日期、結束日期，以及 25 籤售完後是否補第二套。

獎品庫存

上衣、背心、五分褲、Bratop、褲子與風衣需先確認門市可兌換數量。

Pad 權限

確認誰可以開新一套籤、誰可以按已兌換、是否需要每日結算表。

現場物料

需要獎項板、活動規則立牌、Pad 抽籤頁、POS 商品名稱與店員 SOP。
