常見問題

主理人最常問的幾件事

這個方案限哪些教室？

不限教室類型 —— 只要是女性為主的運動空間都適合，包含瑜伽、皮拉提斯、健身房、舞蹈、低衝擊、被動式伸展、水中瑜伽等。標準是「你的學員會穿運動內衣 + 緊身褲」就行。方案僅限教室開幕 1 個月內訂購。

學員可以自己挑款式跟尺寸嗎？

可以，連結內品項學員都能任選：

· 運動內衣 → 從 運動內衣全系列 挑款式 + 尺寸
· 緊身褲 → 從 緊身褲完整品項 挑款式 + 布料 + 長度 + 尺寸

教室收齊學員的選擇後，整批告訴 ASify 客服一次出貨。

學員拿到後不合身怎麼辦？

由教室統一聯繫 ASify 客服處理換貨。寄回的運費由教室負擔，重新寄出的運費由 ASify 負擔。市場上很少有 B2B 方案做到這層保障，這是 ASify 給教室的安心承諾。

為什麼一定要包進開幕方案？

因為「2400 元買到」跟「收到 4400 元的禮物」，是兩種完全不同的感受：

· 如果學員直接以 2400 元購買 → 心裡的反應是「撿到便宜」
· 如果學員收到的是市值 4400 元的整套禮物（包在開幕方案裡）→ 心裡的反應是「被尊榮對待，這間教室有價值」

開幕第一個月要的是後者。尊榮感、價值感才會把學員留下來，撿便宜的記憶很短，留不了人。

出貨需要多久？怎麼付款？

付款完成後 3 個工作天內出貨。付款方式：現金 或 轉帳匯款。教室告訴我們開幕日，我們從那天回推安排。

想先少量試訂可以嗎？

配合開幕優惠才有 1 組起訂的彈性，未滿 5 組要負擔運費（郵寄 75 元 / 超商 65 元）。
平常 ASify 沒有針對教室零售一兩組，這個彈性只在開幕方案存在。
